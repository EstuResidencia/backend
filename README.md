## EsTuResidencia Back-end

EsTuResidencia es la empresa que acompaña a los estudiantes foráneos para obtener arrendamientos a precios económicos y manteniendo la seguridad que se requiere para sentirse seguro y tener una buena vida académica.

In backend's team we work in the dark to praise the light. We ensure fast connection times, altogether with an comprehensible intern system, due for it's modularity and elasticity, which will allow us to keep growing.
